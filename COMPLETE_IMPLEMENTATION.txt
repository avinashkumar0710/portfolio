Solution:
  1. Check MySQL is running in XAMPP Control Panel
  2. Verify database exists: portfolio_admin
  3. Check credentials in config.php
  4. Ensure MySQL user has proper permissions

Problem: "Table doesn't exist"
Solution:
  1. Re-visit http://localhost/portfolio/setup.php
  2. Check for error messages in setup output
  3. Verify database was created
  4. Check MySQL user can CREATE TABLE

Problem: Login doesn't work
Solution:
  1. Press F12 to open browser console
  2. Check for JavaScript errors
  3. Verify api/auth.php exists
  4. Check PHP error logs
  5. Try hard refresh: Ctrl+Shift+R

Problem: Setup.php not working
Solution:
  1. Check PHP is enabled
  2. Verify MySQL extension is installed
  3. Check error logs
  4. Ensure config.php is readable

Problem: Sessions expire too quickly
Solution:
  1. Check SESSION_LIFETIME in config.php
  2. Verify server time is correct
  3. Check PHP session configuration
  4. Look at database for expired records
